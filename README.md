# C-IPL

> C-like language with native list suport

## Usage

```shell
./tradutor.out tests/{input}.cipl (> {output}.txt)?
```

Running valgrind to check for memory leaks

```shell
make mem_check
```

## Installation

```shell
sudo apt install flex gcc make
make
```
