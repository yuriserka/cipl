# C-IPL

> C-like language with native list suport

## Usage

```shell
./tradutores.out < tests/<input>.cipl > tokenized_<output>.txt
```

Running valgrind to check for memory leaks

```shell
make mem_check
```

## Installation

```shell
sudo apt install flex bison gcc make autoconf
make
```
